# ZCode 产品功能与软件设计说明（含 UI 纯文本描述）

官网地址：[https://zcode.z.ai/](https://zcode.z.ai/)

## 一、产品概述

Z Code（官网：[https://zcode.z.ai/](https://zcode.z.ai/)）是智谱 AI 推出的全新**智能体开发环境（Agentic Development Environment，ADE）**，与传统以手动代码编辑为核心的 IDE 不同，Z Code 将 AI Agent 作为开发流程的主角，通过整合主流 AI 编程工具，实现了 "对话即编程" 的全新开发范式，让开发者仅需通过自然语言描述需求，即可驱动 AI 完成从编码、调试到预览的全生命周期开发任务。

Z Code 的核心目标是打破传统 AI 编程 CLI 工具的碎片化壁垒，将 Claude Code、Codex、Gemini、GLM Coding 等分散的 AI 编程能力整合到统一的可视化平台中，降低 AI 编程的使用门槛，同时通过多 Agent 协作机制，大幅提升复杂开发任务的效率。

## 二、核心功能

### 1. 多 Agent 集成与无缝切换



* 支持智谱 GLM Coding、Claude Code、OpenAI Codex、Google Gemini 等主流 AI 编程工具的无缝接入

* 统一的 API Key 管理，支持一键切换不同 Agent，上下文自动共享

* 支持自定义兼容 OpenAI/Anthropic 协议的模型服务（含本地私有模型）

### 2. 自然语言对话式开发



* Agent Chat 交互：自然语言描述需求，AI 自动理解项目结构并完成开发任务

* 思考模式（Think Mode）：动态开启深度思考能力，前置推演需求分析与方案设计

* 自动读取项目文件内容，理解遗留系统业务逻辑

### 3. 精细化权限管控



| 权限模式                     | 行为说明             | 适用场景      |
| ------------------------ | ---------------- | --------- |
| Always Ask（始终询问）         | 每一步操作需用户手动确认     | 新手开发、敏感模块 |
| Accept Edits（接受编辑）       | 自动允许文件修改，高危操作需确认 | 常规业务开发    |
| Plan Mode（计划模式）          | 仅生成方案不执行，用户确认后启动 | 架构设计、大型重构 |
| Bypass Permissions（绕过权限） | 自主执行所有操作         | 快速原型开发    |

### 4. 会话级版本管理



* 对话历史与代码变更深度绑定，生成直观版本时间线

* 支持历史会话重跑、一键版本回滚

* 内置 Git 可视化操作

### 5. 一体化全流程开发环境



* 文件管理面板、内置终端、实时预览浏览器一体化整合

* 多工具数据联动，无需跨窗口切换

### 6. 可视化配置与多任务并行



* 图形化 Agent 配置面板，无需手动修改配置文件

* 多 CLI 实例并行，会话级独立模型配置

### 7. 跨设备远程开发



* 移动端扫码直连桌面端开发环境

* 移动控制台实时查看进度、下达指令

### 8. MCP 开放生态



* 内置联网搜索、视觉理解能力

* 支持自定义 MCP 服务扩展 Agent 能力

## 三、软件设计说明

```
┌─────────────────────────────────────────┐

│ 交互层：桌面端UI + 移动端协同界面       │

├─────────────────────────────────────────┤

│ 编排调度层：Agent调度与核心逻辑层       │

│  - 任务编排、上下文管理、权限控制        │

│ 工具集成层：统一适配层                  │

├─────────────────────────────────────────┤

│ 模型服务层：多模型接入层                │

└─────────────────────────────────────────┘
```

### 2. 核心模块设计



* 全局上下文感知模块：自动扫描项目结构、分析依赖关系

* 权限管控模块：操作拦截与校验，平衡效率与安全

* 会话版本管理模块：对话与文件版本绑定，细粒度版本控制

* 一体化开发模块：多工具整合与数据联动

### 3. 关键设计理念



* 以 Agent 为中心的设计：用户从代码编写者转变为需求决策者

* 低门槛能力集成：可视化操作替代复杂 CLI 配置

* 效率与安全平衡：可配置权限模式适配不同场景

* 开放生态设计：支持 MCP 协议与自定义模型接入

## 四、UI 纯文本详细描述（Plain Text）

### 1. 整体视觉风格与布局结构



* **风格定位**：极简现代、开发者友好，低视觉噪音设计，以功能性为主，配色以深灰 / 黑色为底（暗模式优先），代码区用高亮色彩区分语法，交互元素用低饱和度蓝 / 绿色作为强调色，无冗余装饰。

* **整体布局**：采用 "左侧导航 + 中间核心工作区 + 右侧辅助面板" 的三栏式布局，支持拖拽调整面板宽度，所有面板可折叠 / 展开，重启后自动恢复上次布局状态（含侧边栏折叠状态）。

* **响应式适配**：支持 13 寸笔记本（最小宽度 1280px）与移动端（横屏 / 竖屏），断点自动切换：桌面端三栏布局 → 平板端双栏布局（隐藏左侧导航，可通过滑动呼出） → 移动端单栏布局（面板堆叠显示，支持上下切换）。

### 2. 顶部导航栏（固定置顶，全局可见）



* **左侧区域**：


  * ZCode 品牌 Logo（白色文字 + 简约图形标识，无渐变 / 阴影）

  * 项目名称显示（可点击修改，支持中英文输入）

  * 环境切换下拉菜单（本地 / 远程开发，带图标区分：本地 = 硬盘图标，远程 = 云朵图标）

* **中间区域**：


  * 搜索框（占位文本 "搜索项目文件 / 命令 / 文档"，支持模糊搜索，输入时实时显示匹配结果，搜索结果高亮关键词）

  * 快捷键提示（搜索框右侧显示 "⌘+K 快速搜索"，浅灰色小字）

* **右侧区域**：


  * 权限模式切换按钮（显示当前模式名称，如 "Accept Edits"，点击弹出下拉菜单选择四种模式，按钮背景色随模式变化：Always Ask = 橙色，Accept Edits = 绿色，Plan Mode = 蓝色，Bypass = 灰色）

  * Agent 切换下拉菜单（显示当前使用的 Agent 名称，如 "GLM Coding"，带模型图标，点击切换其他集成 Agent）

  * 通知中心图标（铃铛形状，有未读通知时显示红色小点）

  * 用户头像（圆形，默认显示用户名首字母，点击弹出个人设置、API Key 管理、退出登录选项）

  * 窗口控制按钮（最小化 / 最大化 / 关闭，macOS 风格，无文字仅图标）

### 3. 左侧导航栏（可折叠，宽度 240px-320px 可调）



* **层级结构**：

1. 顶部标题 "项目导航"（14px 灰色文字，居中显示）

2. 项目文件树（核心模块）：

* 以文件夹 / 文件图标区分类型（文件夹 = 灰色文件夹图标，HTML = 红色，JS = 黄色 JS 图标，CSS = 蓝色 CSS 图标，其他文件 = 灰色文件图标）

* 支持展开 / 折叠（点击文件夹前箭头切换），文件名称左对齐，选中文件时背景色变深灰，显示蓝色左侧边框

* 右键菜单支持 "新建文件 / 文件夹" "删除" "重命名" "复制路径" 等操作

1. 任务列表（二级模块）：

* 标题 "当前任务"（12px 灰色文字，带小对勾图标）

* 显示当前正在执行的开发任务（如 "创建智能五子棋游戏"），文字左对齐，带进度条（蓝色填充，显示完成百分比）

* 下方显示历史任务列表（灰色文字，点击可重新激活任务）

1. 底部快捷入口：

* "设置"（齿轮图标 + 文字）

* "文档"（书本图标 + 文字）

* "帮助"（问号图标 + 文字）

* 所有入口为图标 + 文字组合，左对齐，hover 时背景色变浅灰

### 4. 中间核心工作区（占比最大，动态切换内容）

工作区为标签页式设计，支持多标签切换（点击标签切换，右键关闭标签），每个标签对应一个工作模块：

#### （1）Agent Chat 标签（默认打开）



* **上半部分：对话区**（占比 70%）：


  * 左侧：Agent 头像（圆形，对应当前 Agent 图标）+ 名称（灰色文字）

  * 右侧：对话气泡（Agent 回复为浅灰色背景，白色文字；用户输入为蓝色背景，白色文字）

  * 对话内容支持代码块显示（黑色背景，语法高亮，带复制按钮）、列表、链接（蓝色下划线，点击跳转）

  * 支持 "思考模式" 显示：Agent 思考时显示 "正在分析需求..." 文字，带旋转加载动画（灰色小圆圈）

* **下半部分：输入区**（固定底部，高度 80px）：


  * 输入框（占位文本 "描述你的开发需求，如：创建一个智能五子棋游戏..."，支持多行输入，自动换行）

  * 输入框下方：左侧显示 "思考模式" 开关（灰色滑块，开启后变为蓝色），右侧显示发送按钮（蓝色文字 "发送"，点击发送，支持快捷键 ⌘+Enter）

  * 附加功能按钮：文件上传（纸张图标）、引用文件（文件夹图标）、清空对话（垃圾桶图标），均为灰色图标，hover 时变蓝色

#### （2）代码编辑标签（Agent 生成代码后自动打开）



* **顶部**：文件路径显示（浅灰色文字，如 "gomoku-ai/index.html"）+ 保存按钮（磁盘图标，未保存时显示红色小点）

* **代码编辑区**：


  * 左侧行号（浅灰色文字，右对齐）

  * 中间代码区（黑色背景，语法高亮：HTML 标签 = 蓝色，CSS 属性 = 绿色，JS 关键字 = 紫色，字符串 = 黄色）

  * 右侧折叠 / 展开按钮（点击折叠代码块）

  * 支持自动缩进、代码提示（输入时实时显示可能的代码片段，灰色文字，选中后自动补全）

  * 错误提示：语法错误处显示红色波浪线，hover 时显示错误说明（黄色背景，黑色文字）

#### （3）终端标签（内置终端，可手动打开）



* **顶部**：终端名称（灰色文字 "本地终端"）+ 清除按钮（垃圾桶图标）

* **终端区**（黑色背景，白色文字）：


  * 支持命令输入（光标闪烁，输入后按回车执行）

  * 执行结果实时显示（成功信息为绿色文字，错误信息为红色文字）

  * 支持拖拽调整终端高度，resize 后自动重绘，无卡顿

  * 底部吸附逻辑：窗口 resize 时终端始终固定在标签页底部，不偏移

#### （4）预览浏览器标签（前端开发时自动打开）



* **顶部**：地址栏（灰色背景，显示预览地址 "[http://localhost:3000](http://localhost:3000)"，带刷新按钮）+ 设备切换下拉菜单（手机 / 平板 / 桌面图标，点击切换视口）

* **预览区**（占比 90%）：


  * 实时显示代码运行效果（如五子棋棋盘、网页界面）

  * 支持元素选择器：点击预览区元素，自动定位到代码编辑标签对应的代码位置（代码行高亮显示）

  * 响应式预览：切换设备后，预览区自动调整尺寸，显示对应设备的布局效果

  * 内置开发者工具：右键点击预览区，弹出 "检查元素" 选项，点击后右侧显示 DOM 树、样式面板（类似 Chrome 开发者工具）

### 5. 右侧辅助面板（可折叠，宽度 300px-400px 可调）

### 7. 特殊 UI 元素描述



* **空状态**：无项目 / 无对话 / 无搜索结果时，显示居中提示（灰色文字 + 对应图标，如无项目时显示 "暂无项目，点击创建或导入" + 文件夹图标）

  * 居中显示，黑色背景，白色文字，宽度 500px，高度 300px

## 五、适用场景



1. 快速原型开发：自然语言描述需求，Agent 快速生成 MVP

2. 遗留系统维护：Agent 自动理解代码逻辑，辅助修改与 bug 修复

3. 教学与结对编程：对话过程形成开发文档，适合新手学习

4. 全栈小项目开发：一体化环境支持前后端全流程开发

5. 复杂任务并行处理：多 Agent 并行处理多个开发任务

## 六、总结

Z Code 作为新一代智能体开发环境，通过 AI Agent 为核心的设计、一体化工具整合、精细化权限管控，重新定义了 AI 时代的开发方式。其 UI 设计以开发者效率为核心，极简风格减少干扰，三栏式布局与标签页设计支持多任务并行，响应式适配满足不同设备需求，让开发者无需关注复杂工具配置，专注于需求与创意实现。

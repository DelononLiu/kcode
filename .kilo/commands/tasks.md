---
description: 查看任务完成状态，列出未完成、重要/紧急任务
---

用 Read 工具读取 `TASKS.md`，分析全部任务并输出。

## 分析要求

1. **概览**：总任务数 / 已完成数 / 未完成数 / 完成率
2. **未完成任务**（按 Phase 分组排列）：
   - 任务编号 + 标题
   - 当前状态表记（⬜ 未开始 / 🔍 调研中 / 📋 已调研 / 🛠️ 实现中）
   - 优先级（如有）
3. **重要紧急任务**：未完成且标注了 P0 优先级的任务单独高亮
4. **建议**：哪些任务应该优先推进，给出简单理由

## 注意

- 仅汇总表格行（`| PX-XX | ... |` 格式）的任务，忽略独立详情区块中的任务（它们已包含在表格中）
- 如果扫描发现某 Phase 全部已完成，输出一行 `✅ 全部完成` 即可
